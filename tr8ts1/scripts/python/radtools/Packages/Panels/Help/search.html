Search GUI Help

This screen controls the functionality of the search results page. 

 

Initial Defaults:

  •     25 items per page
  •     headword searching off
  •     Sorted Search ordered by Title Ascending

 

Each of the above items can be changed.  Keep in mind that a sorting search does not give you relevance ranking, it orders it by the field you specify. If relenvance ranking is required, please selected Ranked as your sorting type.

 

Preview:

    The preview button will launch a web browser and bring you to the search results listing.  If the current look is good then click the save button and the various templates will get generated for this product.

 

Cancel:

    At any time when making selections it is possible to "rollback" your changes to the last good state (the state at which the GUI was opened) by clicking the Cancel button.

 

Save:

    Once you are satisfied with the way the templates look, clicking save will commit your changes to the product.
