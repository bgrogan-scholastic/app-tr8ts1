Here is where most of the decisions are made about the look and feel of the RAD title.

Article Template - You have to select an article shell to start with. All of the other assets will fit into this template. They are not listed by filename, but rather by a description of the template.

Browse type - There are three browse types: Alphabetical, Subject, and Hierarchical (where the whole browse is structured like a tree, unlike Subject browse, which is more like a filing cabinet.)

Assets Available - If you check the accompanying box, the asset in question will be available in this product.

Button / Thumbnail option - This radio box simply allows you have this asset show up with a number of thumbnails (and possibly a "more" button) or as a button that opens up a new page.

Thumbnail quantity - This allows you to set the amount of thumbnails to show for this asset type.

Priority of this asset - Controls the ordering of this asset type in relation to others. Priority must be unique and sequential for all types.

Pop up Options - What does this thing do?

Preview Button - This allows you to take a look at how your decisions have affected the look of the article. Please note that search might not look the way you want right now. There are more options in the Search Panel.

Cancel Button - Undoes all of your changes.

Save & Close - You're satisfied with the state of things, and you'd like to move on to other portions of this RAD project.
